Dark Mode for Chrome

Privacy Policy

This Privacy Policy explains what data the Dark Mode for Chrome extension uses, what it does not collect, and how extension settings are stored.

Effective date: April 16, 2026 Product: Dark Mode for Chrome

1. Summary

Dark Mode for Chrome is designed to improve readability on websites by applying visual display modes such as Classic Dark, Midnight Black, and Black & White.

The extension does not collect, sell, or share personal data with the developer. It works locally in the browser and stores only the settings needed for the extension to function.

2. Information We Store

The extension stores user preference data in Chrome extension storage so settings can persist between browsing sessions and, if the user has Chrome Sync enabled, across signed-in Chrome instances.

Stored settings may include:

  • whether dark mode is enabled or disabled
  • the selected reading mode, such as Classic Dark, Midnight Black, or Black & White
  • schedule settings such as on and off hours
  • the list of sites where the user has paused the extension

3. Information We Do Not Collect

Dark Mode for Chrome does not collect or transmit the following to the developer:

  • browsing history
  • page content, article text, passwords, or form entries
  • personally identifiable information
  • financial or payment information
  • health information
  • authentication credentials
  • precise location data
  • web analytics or tracking identifiers for advertising

4. Why Permissions Are Needed

<all_urls> host access

This is required so the extension can apply reading styles to websites the user opens in Chrome.

storage

Used to save extension settings such as enabled state, theme mode, schedule, and paused sites.

tabs and activeTab

Used to identify the current tab so the user can pause or resume the extension on the current site and so updates can be applied correctly.

scripting

Used to inject the extension’s content script into web pages so display adjustments can be applied.

alarms

Used to support automatic schedule features for enabling or disabling the extension at selected times.

5. Data Processing and Sharing

Dark Mode for Chrome does not send stored settings or browsing data to the developer’s servers because the extension does not operate a backend service for user data collection.

The extension does not sell user data and does not share user data with third parties for advertising, profiling, or marketing purposes.

6. Third-Party Services

The extension may open third-party websites directly in the browser when a user chooses to visit them, such as the Chrome Web Store listing page. Those websites are governed by their own privacy policies.

7. Data Retention and Control

Stored settings remain in Chrome extension storage until the user changes them or removes the extension. Users can stop all extension data processing by disabling or uninstalling the extension.

8. Children’s Privacy

Dark Mode for Chrome is not directed to children and does not knowingly collect personal information from children.

9. Changes to This Policy

This Privacy Policy may be updated from time to time to reflect product changes, legal requirements, or store policy requirements. The updated version will be posted at this page with a revised effective date.

10. Contact
